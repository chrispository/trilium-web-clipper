Trilium Clipper

Connect directly to your self-hosted Trilium server.

Use a slash-separated note path. For example, Clippings creates a top-level folder, while ideaverse/Clippings creates or reuses that nested path. The date folder is added automatically.

When enabled, the extension creates the destination path and a date book such as 20260714 automatically. Turn this off to use Trilium’s normal clipper inbox behavior.

In Trilium, open Options → ETAPI, create a token, then paste it here. The token stays in this browser’s local extension storage and is sent only to this configured server.
